Projects

The Sustainable Computing Research Group (SCoRe) at UCSC has conducted research covering various aspects of WASN, Security, and Mobile.





FIT4D

FIT4D- A Low Cost Digital Forensic Laboratory for a Developing Country (DFLab4D).

 

Pokuru PC

Pokuru PC- A Low Cost Multi-Terminal.

Mobile ATM

'Mobile ATM' provides standard ATM functions based on the GSM network and mobile phones.

 

S2Net

S2Net is Social Sensor Network Opinion mining tool which provides social information regarding hot news topics.

Tikiri Power

A power monitoring system with enhanced capabilities for power monitoring and controlling using smart phones and tablet PCs.

 

BusNet

BusNet is a sensor network to be built on a public transport system to monitor environmental pollution.

VIP Net

VIPNet can be considered as a tool that can create a virtual IPv4 or IPv6 network with only using a single personal computer.

 

V- Learning

The 'V- Learning' project is an innovative solution for people to access web based learning content through telephones.
